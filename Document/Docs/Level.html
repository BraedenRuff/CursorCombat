Class: Level

Level(scene)

Class Name: Level Description: Manages the game level, including loading level data, spawning shapes, handling interactions with shapes, and determining if a level has been cleared or failed. Expected Inputs: Level data array upon loading a new level. Expected Outputs: Dynamically updates the game state based on user interactions and the game's logic. Called By: GameScene class or any other class responsible for managing game levels. Will Call: Shape classes (Triangle, StealthTriangle, ArmoredTriangle, Square, Architecture, Clear-Friendlies, SlowTime, Intangible) for spawning and managing individual game objects, and callback to scene for some methods.

Methods

checkAllShapes()

Checks each shape within the level to determine if the player's pointer is interacting with it. This method iterates over all shapes and checks for pointer collision based on the shape's geometry. If an interaction is detected, it handles the interaction according to the type of shape. This is used by intangible since if you stop being intangible while over an object it won't register a shape interaction
